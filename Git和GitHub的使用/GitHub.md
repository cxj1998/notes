#### 	2.远程仓库的两种访问方式

​		①**HTTPS**：**零配置，**但每次访问仓库时，需要重复输入GitHub的账号和密码才能访问成功

​		②**SSH**：**需要进行额外的配置，**配置成功后，每次访问仓库时，不需要输入GitHub的账号和密码

#### 	（1）使用HTTPS访问仓库：

​	**更新仓库文件：**

```

```
